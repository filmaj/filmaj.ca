PhoneGap Was Born

  • 2008: More features for iPhone (contacts, media, camera)
  • 2009: Android SDK released late 2008, PhoneGap Android support lands
  • Other platforms came too!
  • BlackBerry
  • Symbian
  • webOS

How It Works

  • PhoneGap is a dressed-up browser
  • Using JavaScript, we can access any bit of device functionality
  • PhoneGap is a native app "shell"

shell per platform

Why JavaScript?

  • Reminder: PhoneGap = browser
  • JavaScript is the language of the web
  • It is everywhere:
  • Embedded in tiny electronic devices
  • On the server (node.js)

PhoneGap is a Native App

  • The PhoneGap "shell"
  • PhoneGap is a native app for each platform

Why Use PhoneGap?

  • Write once, run on five platforms
  • More coming soon!
  • MIT licensed. Read: free and open.

PhoneGap Build

  • A service where you submit your HTML+CSS+JavaScript, it builds your app in PhoneGap, and spits out several native apps back to you
  • You don't need to setup your development environment
  • No more downloading SDKs
  • You don't even need a Mac to build iPhone apps!
